## 搬瓦工如何搭建网站：完整步骤

拿到 VPS 之后，建站的流程大概是这样的：

**第一步：准备好域名**

域名和 VPS 要分开买。选一个域名注册商，买好你想要的域名，然后在 DNS 配置里添加一条 A 记录，把这个域名指向你搬瓦工 VPS 的 IP 地址。IP 在 KiwiVM 控制面板里能看到，登录后台之后首页就有。

DNS 解析生效一般需要几分钟到几小时不等，这段时间先去配置服务器环境。

**第二步：SSH 连接 VPS**

用你的操作系统自带终端（Mac/Linux 用 Terminal，Windows 用 PowerShell 或者 PuTTY），输入：


ssh root@你的VPS_IP -p 端口号


端口号在 KiwiVM 面板里的「SSH Port」那栏能找到，不是默认的 22。密码是购买后系统邮件里发给你的，首次登录建议改掉。

**第三步：选一种建站环境**

搬瓦工 VPS 上建站，环境部署主要有两种方向：

- **宝塔面板（推荐新手）**：有可视化界面，点点按钮就能装好 Nginx + MySQL + PHP，不用敲命令行。在终端里跑一条宝塔的安装命令，安装完成后浏览器打开面板地址，登录进去选「LNMP」套件安装即可。整个过程十几分钟完成，适合不熟 Linux 的人。

- **LNMP/LAMP 一键包**：全命令行操作，更轻量，资源占用少。内存 1GB 的 VPS 推荐这个，Nginx 比 Apache 省内存，跑小站不会卡。一条 `wget` 命令下载安装包，按提示选好 MySQL 和 PHP 版本，等安装完成就好。

两条路选一个，不用两个都装。新手从宝塔开始省心很多。

**第四步：添加网站和数据库**

以宝塔面板为例，进去之后点「网站」→「添加站点」，填写你的域名，同时勾选创建数据库，记好数据库名、用户名和密码，后面 WordPress 安装时要用。

这一步会自动帮你建好 Nginx 虚拟主机配置文件，不需要手动写。

**第五步：上传 WordPress 并安装**

去 WordPress 官网下载最新版安装包，解压之后通过 SFTP 或者宝塔面板自带的文件管理器，把所有文件上传到你的网站根目录下（通常是 `/www/wwwroot/你的域名/`）。

上传完之后，浏览器打开你的域名，WordPress 的安装引导界面就会出来。按提示填写之前记好的数据库信息，设置管理员账号和密码，点「安装」。

大概两分钟，网站就跑起来了。

**第六步：配置 SSL 证书**

没有 HTTPS 的网站，浏览器会显示「不安全」，用户信任感差，搜索排名也受影响。宝塔面板里可以直接申请免费的 Let's Encrypt 证书，点几下就装好，之后自动续期，不需要操心。

装好证书之后，把 HTTP 强制跳转 HTTPS 的开关打开，站点就配置完了。

## 搭建网站后需要做的几件事

网站跑起来只是第一步。用下来觉得以下几件事比较要紧：

- **固定链接设置**：WordPress 后台 → 设置 → 固定链接，改成「文章名」格式，对 SEO 友好，URL 也好看一些。
- **安装缓存插件**：WP Super Cache 或者 W3 Total Cache，搬瓦工的 SSD 读写速度不慢，配上缓存之后页面打开速度会更快。
- **定期备份数据**：搬瓦工 KiwiVM 后台自带自动备份功能，免费的，默认开着。另外可以手动打快照，要迁移机房或者更新系统前打一个，出问题能直接恢复。
- **开启 BBR 加速**：这是 Linux 内核里的 TCP 拥塞控制算法，开了之后网站数据传输效率更高，命令行开启很简单，搬瓦工的 KiwiVM 面板里也可以一键启用。

**Q：搬瓦工 VPS 可以同时跑多个网站吗？**

可以。LNMP 或者宝塔面板都支持一台 VPS 上建多个独立站点，每个站点独立的域名、根目录和数据库。内存和 CPU 够用的情况下，跑几个站没有问题。
